Connect to NocoDB to automatically sync submitted inspections.

Leave as default for NocoDB Cloud.

In NocoDB, open your table → Details tab → copy the Table ID.
NocoDB table columns required:
Address · Neighborhood · Inspector · Inspection Date · Submitted At · Structural Damage · Historical Fabric · Facade Condition · Front Door Condition · Front Padlock · Front Windows · Rear Door Condition · Rear Padlock · Rear Window Condition · Roof Condition · Electrical Line Location · Electrical Line Elevation · Sewer Line Location · Water Line - City Side · Water Line - House Side · Water Meter Present · Back Yard Condition · Hazards to Pedestrians · Hazard Description · Squatter Signs · Squatter Notes · Action Items · Notes · App Record ID

All columns should be Single line text type, except Back Yard Condition, Squatter Signs, Action Items, Hazard Description, and Notes which should be Long text.
